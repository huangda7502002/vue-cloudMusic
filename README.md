# 技术栈

> vue2.0 + vue-router + webpack + vuex + axios

> better-scroll (移动端滚动插件)

> fastclick (解决移动端click事件300ms延迟)

> vue-lazyload (图片懒加载)

> 后端采用基于node的网易云api接口

> 目前主要完成音乐播放和视频的播放, 搜索功能正在开发中.
